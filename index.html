Déploiement Automatique dans le Cloud

Projet de Fin d'Études

Ce projet consiste à mettre en place une chaîne CI/CD simple pour automatiser le déploiement d'une application web dans le cloud.

Objectif

L'objectif est de publier automatiquement un site web après chaque modification du code source.

Technologies utilisées

  • GitHub
  • GitHub Actions
  • GitHub Pages
  • HTML

Pipeline CI/CD

  1. Modification du code source
  2. Déclenchement automatique de GitHub Actions
  3. Vérification du projet
  4. Déploiement automatique sur GitHub Pages
